# 马小的训练计划 V3.3

13周力量与运动表现计划，支持多Round连续训练、跨轮战绩和纪录奖励。

## 训练结构

- 每周四练：A1、B1、A2、B2。
- W4肌肥大减载，W9力量重置减载，W13次极限复测。
- B1为蹲主导：全蹲、哑铃RDL、保加利亚分腿蹲、内收肌。
- B2为髋铰链主导：传统硬拉、前蹲、反向箭步蹲、腿弯举。
- 下肢日可选一个器械收尾；疲劳状态自动隐藏。

## 数据规则

- `schemaVersion: 4`，`planRevision: 3.3-multi-round`。
- 正式存储键为`mx_plan_v3_rounds`，不读取旧正式版或预览版数据。
- 到达W13后可手动开启下一Round；XP、纪录、趋势、最高连胜和历史记录保留。
- 导入导出包含全部Round，重复导入按唯一ID去重。
