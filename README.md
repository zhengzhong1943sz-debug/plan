# 马小的训练计划 V2 · Preview

13 周力量与网球运动表现训练计划预览版。每周四次训练；每次包含一个必做腹肌动作和一个可选核心动作，直接手臂孤立训练与独立核心日已移除。

## 数据边界

- 预览数据使用 `mx_plan_v2_preview`，不会覆盖正式 `/plan/` 的训练记录。
- “复制旧版记录到预览”只读取 `tp_v13`，原数据不会被删除或改写。
- 正式替换前需给旧仓库创建 `v1-legacy` 标签和备份分支。

## 运行

使用任意静态文件服务器打开本目录。计划数据位于 `data/plan.json`，页面逻辑位于 `assets/app.js`。
