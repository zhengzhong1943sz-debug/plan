# 马小的训练计划 V3.1 · Preview

13周力量与通用运动表现训练计划预览版。每周四练，包含阶段主项、低量爆发启动、动作详情、个人纪录奖励和按时间查询的力量趋势。

## V3重点

- W1–4肌肥大、W5–8传统硬拉力量、W9卸载、W10–12运动表现、W13次极限复测。
- B1、B2分别安排肩中束与后束超级组；地雷管仅作为备选。
- 爆发启动可按状态跳过，不影响通关和XP。
- 技术合格组支持重量、固定重量次数和估算1RM三类纪录；同一动作每次训练最多奖励30 XP。
- 战绩面板支持按动作、变式、时间和指标查询；无速度传感器时不展示虚假速度。
- B1调整为高杠全蹲主导，搭配哑铃罗马尼亚硬拉；B2调整为传统硬拉主导，搭配前蹲。
- 反向箭步蹲和腿弯举在肌肥大、运动表现阶段必做，在硬拉力量与重置阶段转为可选。

## 数据边界

- 预览数据继续使用独立键 `mx_plan_v2_preview`，并在本地自动迁移至`schemaVersion: 3`，不会覆盖正式`/plan/`记录。
- “复制旧版记录到预览”只读取 `tp_v13`，原数据不会被删除或改写。
- 正式替换前需给旧仓库创建 `v1-legacy` 标签和备份分支。

## 运行

使用任意静态文件服务器打开本目录。计划数据位于`data/plan.json`，页面逻辑位于`assets/app.js`，可测试的纪录与迁移算法位于`assets/logic.mjs`。
